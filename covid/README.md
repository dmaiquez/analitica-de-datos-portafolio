# Analítica de Datos Globales: COVID-19, Vacunación, Demografía e Indicadores Económicos

## Descripción

Proyecto de analítica de datos desarrollado para la enseñanza de Big Data, Business Intelligence y Data Warehouse mediante la integración de múltiples fuentes de información relacionadas con COVID-19, vacunación, demografía mundial e indicadores de desarrollo económico.

El proyecto fue diseñado siguiendo una metodología progresiva de aprendizaje basada en proyectos, donde inicialmente se construyen modelos estrella independientes y posteriormente se integran en una arquitectura de constelación de hechos para el análisis multidimensional de datos.

La solución fue desarrollada en Power BI utilizando conceptos de ETL, modelado dimensional, análisis geoespacial y visualización avanzada.
