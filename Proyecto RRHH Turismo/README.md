
## Proyecto: Analítica de Datos - Contrataciones Sector Turístico

Descripción

Sistema de gestión y análisis de contrataciones desarrollado como proyecto interdisciplinario entre las carreras de Desarrollo de Software y Turismo.

El proyecto fue diseñado tomando como referencia los procesos de una empresa real del sector turístico ecuatoriano dedicada a la organización de tours y actividades turísticas, principalmente en la región amazónica.

La solución permite registrar información de nuevas contrataciones mediante una aplicación web desarrollada con Spring Framework, JSP y MySQL.

Posteriormente, la información es consumida por Power BI Desktop y Power BI Service para la generación de dashboards con actualización automática programada.

## Tecnologías Utilizadas
- Java
- Spring Framework
- JSP
- MySQL
- Power BI Desktop
- Power BI Service
- GitHub

La solución permitió visualizar información de contrataciones en tiempo casi real mediante dashboards conectados a MySQL y publicados en Power BI Service.

La arquitectura implementada facilitó la integración entre aplicaciones transaccionales y herramientas de inteligencia de negocios.

## Nota de Privacidad

Importante:
El proyecto fue inspirado en los procesos de una empresa real del sector turístico. Sin embargo, los datos utilizados en las pruebas y dashboards corresponden a información simulada generada con fines académicos, por lo que no representan información real de empleados ni contrataciones.
