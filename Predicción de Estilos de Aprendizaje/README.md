## Predicción de Estilos de Aprendizaje mediante Inteligencia Artificial
Descripción

Proyecto de investigación enfocado en la identificación y predicción de estilos de aprendizaje en estudiantes de Tecnología Superior en Desarrollo de Software mediante técnicas de Inteligencia Artificial, Machine Learning y Analítica de Datos Educativos.

La investigación combina Learning Analytics, Educational Data Mining, Data Warehouse, generación sintética de datos, Transfer Learning y modelos de clasificación para predecir los estilos de aprendizaje definidos por el modelo de Felder y Silverman.

El estudio fue desarrollado con 76 estudiantes y aproximadamente 3500 respuestas obtenidas mediante el cuestionario de estilos de aprendizaje.

## Objetivo

Identificar los estilos de aprendizaje predominantes en estudiantes de Desarrollo de Software y desarrollar modelos de Inteligencia Artificial capaces de predecir las dimensiones de aprendizaje propuestas por Felder y Silverman.

La arquitectura propuesta integra:

- Redes neuronales MLP.
- Embeddings generados a partir de datasets educativos externos.
- Transfer Learning.
- Generación sintética de datos mediante Variational AutoEncoder.
- Balanceo de clases con SMOTE.
- Optimización mediante RandomizedSearchCV.
- Cuatro modelos Random Forest especializados para cada dimensión del modelo Felder-Silverman.
