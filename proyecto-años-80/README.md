## Proyecto: Analítica de Datos - Casa Abierta Años 80
Descripción:

Sistema de captura y análisis de datos desarrollado para una casa abierta temática de los años 80. La información registrada por los visitantes era almacenada en SQL Server y visualizada mediante dashboards en Power BI con actualización automática cada 5 minutos.

## Rol
Mi participación:

- Diseño de arquitectura de datos.
- Diseño del modelo dimensional.
- Desarrollo de dashboards Power BI.
- Configuración de Power BI Service.
- Configuración de actualizaciones automáticas.
- Supervisión técnica del desarrollo web.
- Integración SQL Server - Power BI.

Proyecto desarrollado junto a estudiantes de la carrera de Desarrollo de Software, donde participé como docente líder y responsable de la arquitectura de datos y analítica.
